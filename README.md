# Apple Store — учебный Django-проект интернет-магазина

Учебный командный проект по дисциплине «Web / Backend-разработка».  
Цель проекта — продемонстрировать работу Django, серверной логики, базы данных, CRUD-операций и командной разработки с использованием Git и GitHub.

Проект выполнен командой из трех человек с разделением ролей и работой через ветки и pull-request’ы.

---

## Описание проекта

Apple Store — упрощенный интернет-магазин техники Apple с каталогом товаров, поиском, фильтрацией, корзиной и оформлением заказов.

Приложение реализует полный пользовательский сценарий:
- просмотр каталога товаров;
- поиск, сортировку и пагинацию;
- регистрацию и авторизацию пользователей;
- работу с корзиной;
- оформление заказов;
- хранение данных в базе.

---

## Функциональные возможности

### Основной функционал (CRUD)
- Товары:
  - создание, редактирование и удаление (через Django Admin);
  - просмотр каталога и карточки товара.
- Заказы:
  - создание заказа;
  - просмотр заказов пользователя.
- Пользователи:
  - регистрация;
  - вход и выход;
  - профиль пользователя.

### Дополнительный функционал
- поиск по товарам;
- фильтрация и сортировка каталога;
- пагинация;
- корзина (добавление, удаление, изменение количества товаров);
- авторизация и разграничение доступа;
- пользовательский интерфейс на Bootstrap;
- адаптивная верстка.

---

## Архитектура проекта

Проект разделен на логические Django-приложения:

- shop — каталог товаров, категории, поиск, фильтрация, пагинация;
- cart — логика корзины;
- orders — оформление заказов (Order, OrderItem);
- accounts — регистрация, авторизация, профиль пользователя;
- templates — HTML-шаблоны;
- static — статические файлы.

Архитектура построена с разделением ответственности между приложениями и переиспользованием шаблонов.

## Командная работа и вклад участников

Разработка велась с использованием GitHub, отдельных веток и pull-request’ов.

- Участник 1 (тимлид, backend-core):
  - инициализация проекта;
  - архитектура и модели;
  - каталог товаров;
  - поиск, фильтрация и пагинация;
  - code review;
  - сбор веток и оформление README.
- Участник 2 (пользовательская логика):
  - регистрация, логин и логаут;
  - профиль пользователя;
  - корзина;
  - оформление заказов;
  - модели Order и OrderItem;
  - права доступа и обработка ошибок.
- Участник 3 (frontend и UI):
  - base.html, шапка и футер;
  - подключение Bootstrap;
  - карточки товаров;
  - страницы корзины, checkout и профиля;
  - адаптивность и финальная полировка интерфейса.

На защите проекта каждый участник объяснял свою часть.

## Установка и запуск проекта

### Требования
- Python 3.10 или выше;
- Django 4 или выше.

### Установка
```bash
git clone https://github.com/TemirlanKoybaev/apple-store.git
cd apple-store
python -m venv .venv
source .venv/bin/activate   # Linux / macOS
.\.venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

### Запуск
```bash
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

Приложение будет доступно по адресу:
http://127.0.0.1:8000/

Админ-панель:
http://127.0.0.1:8000/admin/
